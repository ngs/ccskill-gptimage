## 背景

現在の `SKILL.md` / `SKILL.ja.md` の「Prompt Design Guide」節は、主に `gptimageguide.md`(社内まとめ)を再構成したもの。`gptimageguide.md` 自体は元々 Web Claude 経由の情報を含んでおり、issue #014 で一部に誤情報があったことが発覚し訂正済み。

**現状の限界**:
- OpenAI 公式ドキュメントを **網羅的に** ベストプラクティスとして抽出していない
- 特に重要な OpenAI Cookbook の画像生成プロンプティングガイドが未参照
- このため Claude が SKILL.md を読んでプロンプトを構築する際の品質が、公式ベストプラクティスの恩恵を最大限に受けていない

姉妹スキル `ccskill-nanobanana` では公式情報に基づく best practice をスキル中に文章化しており、本スキルでも同等のクオリティに揃える必要がある。

---

## 目的

公式一次資料を網羅的に調査し、その知見を SKILL.md / SKILL.ja.md に統合することで:

1. Claude が構築するプロンプトの品質を向上(同じ意図でもより良い画像が出る)
2. 世界中のユーザに対してオフィシャルベストプラクティス準拠の体験を提供
3. ドキュメントの根拠が明確になり、将来の保守・引用が容易になる

## 備考

このタスクが完了するまで #004(GitHub 公開)を待つ理由:
- 公開直後にユーザが触る最初のコンテンツが SKILL.md であり、ここの完成度がスキルの第一印象を決める
- 公開後にベストプラクティスを大幅刷新すると、初期ユーザの体験と後続ユーザの体験が乖離する
- 一度公開したリポジトリは履歴が残るため、初回 push の品質に妥協したくない
